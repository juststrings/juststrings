## 🚀 Featured Projects

### 🛡️ FraudGuard — Hybrid Financial Fraud Decisioning Engine
> **AISOC 2025 Hackathon Winner · $750 Prize**

A production-grade, real-time fraud engine that solves the "Precision Paradox." Goes beyond a binary classifier with a **Zero-Trust, 3-layer decision architecture**:

- **Layer 1 — Policy:** Instant blocks for velocity attacks & credit limit breaches
- **Layer 2 — ML:** Random Forest inference (96% recall on 200k transactions, 3× PR-AUC improvement over XGBoost)
- **Layer 3 — Biometrics:** Google MediaPipe facial verification for grey-area transactions

**Stack:** `FastAPI` · `Scikit-learn` · `Polars` · `Prefect` · `Streamlit` · `Render`
**Key Stats:** PR-AUC: 0.363 · Recall: 96% · Net savings simulated: $466,560
